fun GoogleMap.markerDragEvents(): Observable<MarkerDragEvent>
Content copied to clipboard

Creates an Observable that emits events when a marker is dragged.

The created Observable uses GoogleMap.setOnMarkerDragListener to listen to map drag events. Since only one listener at a time is allowed, only one Observable at a time can be used.
